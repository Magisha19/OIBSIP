# Task 1 - Basic Network Scanning with Nmap
## Oasis Infobyte Internship - Cyber Security Track

## What is Nmap?
Nmap is a free tool to discover open ports and services on a network.

## Tools Used
- Nmap 7.98
- Kali Linux

## Scans Performed
- Basic Scan: nmap 127.0.0.1
- Version Scan: nmap -sV 127.0.0.1
- OS Detection: nmap -O 127.0.0.1

## Results
- Port 80 open - Apache HTTP 2.4.66
- OS: Linux 5.0 - 6.2

## Ethical Note
Scanned only my own local machine (127.0.0.1)
